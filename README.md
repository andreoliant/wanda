# wanda

Install git
Install node.js

Then on Terminal:
$:
cd PATH-TO-FOLDER/wanda
git pull origin master
...
cd PATH-TO-FOLDER/wanda/calcio
npm install express
npm install pug
npm install react react-dom
npm install babel-cli@6 babel-preset-react-app@3
npm install sqlite3
...
node server

Then browse to http://localhost:5000
